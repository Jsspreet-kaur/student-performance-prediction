# Student-performance-prediction
This is the machine learning project that predicts "academic performance" based on factor such as study hours, attendance, parental education and other relevant features. This process include collecting the data in the format of .csv file, preprocess data, split data, training the model, predictions are made, evaluate model, save and deploy model. 

**Technologies Used**:
 •Python
 •Pandas
 •NumPy
 •Scikit-learn
 •Matplotlib
 •Seaborn
 •Streamlit

 **How to run the app**(in the terminal)v
     streamlit run SFTnotebookAPP.py

    Note: On the place of the name of the app i saved-- SFTnotebookAPP.py. It will be replaced with the file name saved by the user. 
    streamlit run (app.py)

 **Project Goal**: 
 To develop a predictive model that estimates student performance using historical educational data, helping educators and students to identify factors that influenc academic success.
